## 🚀 Sekilas Tentang Proyek
Sistem ini dirancang untuk memudahkan operasional bengkel dalam mencatat transaksi, mengelola inventaris suku cadang, dan memantau performa bisnis melalui dashboard statistik yang intuitif.

## ✨ Fitur Utama
| Fitur | Deskripsi |
| :--- | :--- |
| **Dashboard** | Statistik pendapatan, stok kritis, dan mekanik aktif. |
| **CRUD Data** | Manajemen pelanggan, kendaraan, dan suku cadang. |
| **Transaksi** | Kalkulasi biaya otomatis & pengurangan stok *real-time*. |
| **Cetak Nota** | Generate invoice rapi dan profesional untuk pelanggan. |
| **Keamanan** | Autentikasi admin dengan enkripsi *password* (Bcrypt). |

## 💻 Teknologi yang Digunakan
* **Frontend:** Bootstrap 5, CSS3, HTML5
* **Backend:** PHP Native
* **Database:** MySQL / MariaDB

---

## 🛠️ Instalasi Lokal
1. **Clone** repositori ini ke folder server lokal (`htdocs` atau `www`).
2. Buat database `db_bengkel_mobil` di MySQL.
3. Import file `db_bengkel_mobil.sql`.
4. Sesuaikan `koneksi.php` dengan pengaturan database Anda.
5. Akses melalui browser: `http://localhost/bengkel-mobil`.
